# 3. Dokumentatsioon

Projektdokumentatsioon on elektripaigaldiste projekteerimise põhiväljund. Kvaliteetne dokumentatsioon tagab projekti arusaadavuse, ehitatavuse ja hilisema hooldatavuse. Käesolev peatükk kirjeldab dokumentatsiooni vormistamise nõudeid ja häid tavasid.
