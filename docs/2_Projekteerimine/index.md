# 2. Projekteerimise põhimõtted

Elektripaigaldiste projekteerimine on süstemaatiline protsess, mis algab lähteülesande analüüsist ja lõpeb tööprojekti valmimisega. Käesolev peatükk kirjeldab projekteerimise etappe, koostööd teiste osapooltega ning riskianalüüsi põhimõtteid.
