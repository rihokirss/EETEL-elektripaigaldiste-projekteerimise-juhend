# 5. Nõrkvool

Käesolev peatükk on **arendamisel**.

Nõrkvoolupaigaldiste projekteerimise juhend käsitleb:

* **Andmeside ja struktureeritud kaabeldus:** Kategooriad, kanalid, testimine
* **Sidevõrgud:** Telefonisüsteemid, intercom, hoone sisevõrk
* **Antennisüsteemid:** TV, raadio, mobiilside
* **Turvasüsteemid:** Videovalve, läbipääsukontroll, valvesignalisatsioon
* **Tulekahjusignalisatsioon (ATS):** Andurid, keskseadmed, evakuatsioon
